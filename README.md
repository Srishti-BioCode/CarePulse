**CarePulse** (also known as Med-Alert) is a lightweight, responsive health-monitoring platform developed as a capstone project for **IIT Patna**. This website is specifically designed to provide senior citizens with easy access to health tracking and emergency features using a clean, accessible interface.

## 🌟 Key Features

*   **Medicine Alert by Alarm:** An automated alarm system that triggers sound/notifications at specific medicine timings to ensure no dose is missed.
*   **Water Tracking System:** A dedicated hydration monitor that helps seniors track their daily water intake and stay healthy.
*   **Emergency Alerts:** Immediate notification triggers for quick medical assistance.
*   **Accessible UI:** Built with high-contrast CSS and large elements for maximum readability.
*   **Real-time Logic:** Dynamic DOM manipulation for health stats and tracking updates.

## 🛠️ Tech Stack

*   **Markup:** HTML5
*   **Styling:** CSS3 (Custom Styles)
*   **Logic:** Vanilla JavaScript (ES6+)
*   **Editor:** VS Code
*   **Backend:** Firebase (For alerts, reminders, and data sync)

## 🚀 How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/carepulse.git](https://github.com/your-username/carepulse.git)
    ```
2.  **Open in VS Code:**
    Open the project folder.
3.  **Launch:**
    Open `index.html` with **Live Server** in VS Code or open it directly in your browser.
